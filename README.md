# Newsletter Email (Laravel)
Простое Laravel-приложение для отправки email-рассылок. Проект создан в образовательных целях.


Стек
Laravel 11

Breeze (Blade)

MySQL

SMTP

## ⚙️ Установка

```bash
git clone https://github.com/Mohabat1/Newsletter-email.git
cd Newsletter-email
composer install
cp .env.example .env
php artisan key:generate
php artisan migrate
php artisan serve
